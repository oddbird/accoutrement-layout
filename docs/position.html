accoutrement-layout » position

@mixin position()

Description

Shortcut for positioning an element, using the common TRBL syntax.

Parameters & Output & Return

$position: null (list)

A shorthand-syntax of positioning values using the common TRBL pattern, e.g. absolute 0 0 3em auto or fixed 0 2%. The position value (absolute, relative, etc) defaults to absolute. Any null values are applied to the side offsets.

{CSS output} (code block)

top, right, bottom, left, and position properties

@error

Unknown positioning value: #{$item}

Examples

scss stretch over the full viewport
.overlay {
  @include position(fixed 0);
}
css compiled
.overlay {
  bottom: 0;
  left: 0;
  position: fixed;
  right: 0;
  top: 0;
}
scss top right offset (absolute is impied)
.overlay {
  // could also use `null` for bottom & left values…
  @include position(2em 2em auto auto);
}
css compiled
.overlay {
  bottom: auto;
  left: auto;
  position: absolute;
  right: 2em;
  top: 2em;
}
