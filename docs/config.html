$breakpoints

scss
$breakpoints: () !default;

Description

Establish a map of named breakpoints.

If you have Accoutrement-Scale installed, you can use the scale syntax for describing size-relationships and adjustments – and even reference $sizes as though they are $breakpoints.

Examples

scss defining breakpoints
$breakpoints: (
  'small': 30em,
  'medium': 45em,
);
scss using accoutrement-scale
$breakpoints: (
  'small': 30em ('convert-units': 'browser-ems'),
  'medium': 'small' ('times': 1.5),
);
